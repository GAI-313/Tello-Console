# get_height コマンド

```get_height()```

　ドローンの相対飛行高度を取得します。

- **get_tof コマンドとの違い**<br>
　***[get_tof コマンド]() では、ToF センサーを用いて精密な飛行高度を取得できます。一方、get_height コマンドでは、気圧計を用いて、離陸地点からの相対的な硬度を取得します。精密なオペレーションを組むのであれば、get_tof が良いでしょう。
