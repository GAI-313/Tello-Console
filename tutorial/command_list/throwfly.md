# throwfly コマンド

```throwfly()```
<br>

　このコマンドは **throwfly（手のひらから離陸）** モードを有効にするコマンドです。このコマンドが実行されると、ドローンはその場で低速でモーターを回転させ、手から優しく投げることでその場で飛行します。

- 手のひら以外でこのコマンドを実行することは推奨されていません。手のひら以外でドローンを離陸させたい場合は、**[takeoff コマンド]()** 、**[motor_start コマンド]()** を使用してください。
