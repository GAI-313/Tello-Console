# rc コマンド

```rc(erlon, elevator, srotol, lador)```
<br>

　このコマンドは、4つの引数を取得します。引数に入れる値は全て **int** で、入力可能範囲は **-100 ~ 100** です。この範囲を超えた値が入ると、自動的に修正されます。

- **第1引数 : erlon**<br>
 エルロン出力として渡します。正の値を渡すと 右移動します。負の値を渡すと 左移動します。
- **第2引数 : elevator**<br>
 エレベーター出力として渡します。正の値を渡すと 前進します。負の値を渡すと 後進します。
- **第3引数 : srotol**<br>
 スロットル出力として渡します。正の値を渡すと 上昇します。負の値を渡すと 下降します。
- **第4引数 : lador**<br>
 ラダー出力として渡します。正の値を渡すと 右旋回します。負の値を渡すと 左旋回します。

## エルロン、エレベーター、スロットル、ラダー とは？
　この4つの名称は、**ドローンのコントロールスティックの取り舵の名前です。** 

- エルロン：機体の左右操作
- エレベーター：機体の前進後進
- スロットル：機体の上昇下降
- ラダー：機体の旋回
